# Changesets

This directory holds [Changesets](https://github.com/changesets/changesets) config and pending changesets.

Run `pnpm changeset` when a change should affect a package's version once packages start being published (Phase 12). See [`config.json`](config.json) for the versioning rules, and the [Changesets documentation](https://github.com/changesets/changesets/tree/main/docs) for command usage.
