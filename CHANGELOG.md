# Changelog

Agentform versions each `@agentform/*` package independently via [Changesets](https://github.com/changesets/changesets) — see [`docs/release-process.md`](docs/release-process.md). Once a package has had at least one release, its own history lives in `packages/<name>/CHANGELOG.md` (or `apps/<name>/CHANGELOG.md`), generated automatically by `pnpm changeset version` from the changesets attached to each change.

No package has been released yet, so no per-package changelog exists yet either. This file exists so that link and convention are documented before the first release, rather than backfilled after.
